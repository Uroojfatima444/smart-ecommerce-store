#  HomeHaven - Home Decor E-Commerce Store

HomeHaven is a modern and responsive home decor and furniture e-commerce website built using HTML, CSS, JavaScript, and Bootstrap.

##  Features

- Product Search
-  Category Filtering
-  Shopping Cart System
-  Increase / Decrease Quantity
-  Remove Products from Cart
-  Local Storage Cart Persistence
-  Product Details Modal
-  Checkout & Payment UI
-  Dark Mode
-  Wishlist Button
-  Responsive Design
-  Modern Home Decor UI

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- Local Storage

##  How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in your browser.
4. Start shopping!
